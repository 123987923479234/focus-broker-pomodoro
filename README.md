# Focus Broker

**当前状态：v0.1.0-alpha，核心任务驱动番茄流程已实现，仍在进行真实长时长验收和数据管理能力补齐。**

Focus Broker 是一个任务驱动的番茄复盘工作台。它不是只展示倒计时的装饰页，而是把任务、计时、休息、复盘和本地统计串成一条可以每天使用的专注流程。

## 产品定位

用任务驱动番茄钟，让每一轮专注都有记录和复盘。

Focus Broker 面向需要持续推进学习、写作、编码、设计和复盘工作的个人用户。当前版本优先保证核心流程可信：先选择任务，再开始专注，完成后记录复盘，并把结果沉淀到今日统计和历史记录中。

## 核心功能

- 任务绑定计时：开始专注前必须选择或创建任务，避免空转计时。
- 番茄循环：支持专注、短休息、长休息和 4 轮循环节奏。
- 有效番茄规则：达到计划时长 80%，且复盘为完成或部分完成，才计入有效番茄。
- 本地复盘：记录完成状态、精力评分、中断次数、原因和备注。
- 统计报告：展示今日结论、有效番茄、专注分钟、完成任务、近 7 天节奏和任务类型分布。
- 专注环境：提供低干扰主题与合成雨声 + 键盘白噪音；背景实验项暂时隐藏，默认使用冷灰蓝低干扰背景。
- 本地数据管理：支持导出历史记录、导入历史 JSON、清空任务、清空历史记录和全部重置。
- 本地保存：任务、设置和历史记录保存在浏览器本地数据中。

## 运行方式

```bash
npm install
npm run dev
```

开发服务器默认访问地址：

```text
http://127.0.0.1:5173/
```

构建并预览生产版本：

```bash
npm run build
npm run preview
```

如果已经构建过，只启动本地静态预览：

```bash
npm run serve
```

生成桌面版相关命令：

```bash
npm run desktop:portable
npm run desktop:dist
```

## 当前限制

- 数据仅保存在本地浏览器或桌面应用环境中，暂不提供账号和云同步。
- 背景切换是实验项，当前主界面不开放多背景选择，避免干扰任务和复盘主线。
- 浏览器通知依赖用户授权；权限被拒绝时会在页面和设置中提示，但仍需更多系统级提醒策略。
- 长时间自然到点、连续四轮长休息、关闭后补录等场景仍需要更多真实使用验收。
- 白噪音为 Web Audio API 合成声音，不包含外部音乐版权素材。
