# v0.1.0-alpha

Focus Broker 是一个任务驱动的番茄复盘工作台，用任务、计时、休息和复盘帮助用户把每天的专注推进变得可见、可记录、可回看。

## 已实现

- 任务绑定计时：开始专注前必须选择或创建任务。
- 番茄循环：支持专注、短休息、长休息和 4 轮循环。
- 有效番茄规则：达到计划时长 80%，且复盘为完成或部分完成，才计入有效番茄。
- 本轮复盘：记录完成状态、精力评分、中断次数、原因和备注。
- 今日报告：展示有效番茄、专注分钟、完成任务、复盘精力和自然语言结论。
- 历史记录：保留每轮专注记录，删除任务不删除历史统计。
- 专注环境：保留低干扰主题和合成雨声 + 键盘白噪音。
- 本地数据管理：支持导出历史、合并导入、覆盖导入、清空任务、清空历史记录和全部重置。
- 通知权限被拒绝时，页面和设置中都会显示轻量提示。
- 极简模式和基础桌面端打包支持。

## 当前限制

- 数据仅保存在本地浏览器或桌面应用环境中，暂不支持账号和云同步。
- 背景切换仍为实验项，主界面暂不开放。
- 通知权限依赖浏览器授权，被拒绝时会提示用户重新开启。
- 真实 25 分钟自然到点和连续 4 轮长休息仍需更多实际使用验收。
- 本地数据导入 / 恢复仍处于 Alpha 阶段，覆盖导入前请先导出现有历史记录备份。
