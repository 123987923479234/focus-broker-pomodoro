# v0.1.0-alpha

Focus Broker 是一个任务驱动的番茄复盘工作台，用任务、计时、休息和复盘帮助用户把每天的专注推进变得可见、可记录、可回看。

## 产品定位

Focus Broker 面向需要持续推进学习、写作、编码、规划和研究任务的个人用户。当前版本是 Alpha，用于验证核心番茄流程、任务生命周期、本地复盘和统计口径。

## 已实现

- 任务绑定计时：开始专注前必须选择或创建任务。
- 番茄循环：支持专注、短休息、长休息和 4 轮循环。
- 有效番茄规则：达到计划时长 80%，且复盘为完成或部分完成，才计入有效番茄。
- 本轮复盘：记录完成状态、精力评分、中断次数、原因和备注。
- 今日报告：展示有效番茄、专注分钟、完成任务、复盘精力和自然语言结论。
- 历史记录：保留每轮专注记录，删除任务不删除历史统计。
- 专注环境：保留低干扰主题和合成雨声 + 键盘白噪音。
- 极简模式和基础桌面端打包支持。
- 本地数据管理：支持导出历史记录 JSON、清空今日任务、清空历史记录和全部重置。

## 当前限制

- 数据仅保存在本地浏览器或桌面应用环境中，暂不支持云同步。
- 背景切换仍为实验项，主界面暂不开放。
- 浏览器通知依赖用户授权，权限被拒绝时已有轻量提示，但仍需更明确的全局引导。
- 本地数据恢复入口仍需补齐。
- 仍需要更多真实任务数据验证 7 天趋势和任务类型统计。

## 已知问题

- 连续 4 轮后的长休息逻辑已在代码中存在，但仍需要真实连续流程验证。
- 浏览器关闭、系统睡眠、通知权限变更等边界场景仍需更多测试。
- 图表在数据量极少时只能提供基础解释，长期趋势判断需要更多样本。
