# Focus Broker v0.1.0-alpha Release Notes Draft

## Release Type

Alpha release. This version is intended for product validation, local testing, and early feedback. It should not be positioned as a finished commercial release yet.

## Product Positioning

Focus Broker is a task-driven Pomodoro workspace for deep focus and daily review. The product priority is task progress, focus timing, break rhythm, review records, and local history.

## Included In This Alpha

- Task-bound Pomodoro flow: users create or select a task before starting focus.
- Focus, short break, and long break cycle support.
- Effective Pomodoro rule based on planned duration and review result.
- End-of-round review with completion status, energy score, interruptions, and notes.
- Today report with effective Pomodoros, focus minutes, completed tasks, and weekly trend.
- Local persistence through browser storage and IndexedDB.
- Low-distraction focus environment with synthetic rain and keyboard sound.
- Desktop packaging scripts for local builds.

## Current Limitations

- Data is local only; there is no account system or cloud sync.
- Notification behavior depends on browser permission.
- Background scene experiments are hidden from the main UI in this alpha.
- Long-running real-world flows still need more validation, especially refresh recovery, four-round cycles, and missed review recovery.
- Audio is generated with Web Audio API and does not include licensed music tracks.

## Suggested Review Checklist

- Add a task, start a focus session, complete a round, submit review, and confirm the task progress updates.
- Confirm an incomplete short session contributes focus minutes but not effective Pomodoros.
- Verify task deletion keeps historical statistics understandable.
- Test desktop, tablet, and 375px mobile layouts.
- Test notification permission denied and approved states.

## Release Decision

Recommended label: v0.1.0-alpha.

Recommended public wording: early alpha for task-driven Pomodoro workflow validation.
